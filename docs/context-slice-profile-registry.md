# Context Slice Profile Registry v0.1

## Purpose

This registry defines the allowed slice profile identifiers for `context.slice`.

## Profiles

### `task_focus`

- description:
  - Generic task-focused sub-context.

### `source_focus`

- description:
  - Preserve source payload emphasis for downstream operations.

### `trace_focus`

- description:
  - Preserve trace-facing details for inspection workflows.

## Current Support

The current implementation supports:

- `task_focus`
- `source_focus`
- `trace_focus`
